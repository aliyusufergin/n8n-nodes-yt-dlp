# İlk yayın yalnızca resmî n8n Docker image Linux x64'ü destekler

İlk yayının platform desteği resmî n8n Docker image üzerindeki Linux x64 ile sınırlı olacak ve selector yalnızca kanıtlanmış x64 platform paketini seçecek. Resmî n8n ve yt-dlp arm64 artifact'leri mevcut olsa da paketleme tek başına gerçek Community Packages kurulumu, queue worker yüklemesi, yt-dlp/FFmpeg/FFprobe çalışması ve lisans source-bundle bütünlüğünü kanıtlamadığından arm64 ancak native E2E sonrası ayrı bir sürümde eklenecek.

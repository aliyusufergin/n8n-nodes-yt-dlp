# Yeni tasarım 0.2.0 olarak yayımlanır

Yeni ana paketin ilk sürümü `0.2.0` olacak. Eski `0.1.0` yeniden kullanılamadığı ve yeni tasarım node sözleşmesi, güvenlik sınırı, output modeli ile paket topolojisini değiştirdiği için patch devamı izlenimi veren `0.1.1` yerine pre-1.0 çizgisinde yeni tasarım sınırını gösteren `0.2.0` seçildi; korunan eski release ve Corresponding Source Bundle değiştirilmeyecek.

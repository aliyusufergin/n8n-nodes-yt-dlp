# Corresponding source

The exact source locations, versions, licenses, and archive digests are recorded in `toolchain-manifest.json`. The Corresponding Source Bundle for version 0.1.0 is attached to the matching GitHub release at:

https://github.com/aliyusufergin/n8n-nodes-ytdlp/releases/download/v0.1.0/n8n-nodes-ytdlp-0.1.0-sources.tar.gz

That bundle includes source snapshots, the build recipe, configuration, patches, and inventory used for these binaries. It remains available for as long as this binary package is distributed.
